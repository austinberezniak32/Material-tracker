How to get your API key
1. Go to console.anthropic.com on any browser
2. Sign up for a free account
3. Go to API Keys in the left menu
4. Click Create Key — name it "SSM Tracker"
5. Copy the key (starts with sk-ant-)
6. Paste it in the field above and tap Save
7. Add a payment method — minimum $5 credit, lasts a long time
